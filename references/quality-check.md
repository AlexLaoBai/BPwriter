## §1 四层检查单

### S层检查（自动化执行）

| # | 检查项 | 阈值 | 检测方式 |
|---|--------|------|---------|
| S1 | 首句含人物/场景/数据/问题？ | 违规=0 | Agent判断 |
| S2 | A类禁用词？ | 出现=0 | 词表匹配 |
| S3 | 超40字句占比？ | ≤10% | 自动计数 |
| S4 | C类连接词密度？ | ≤2次/千字 | 词表匹配 |
| S5 | B类禁用词？ | 出现=0 | 词表匹配 |
| S6 | 标点频率达标？ | 见 `quality-S.md` S6 | 自动计数 |

### P层检查（Critic-L1执行）

| # | 检查项 | 阈值 |
|---|--------|------|
| P1 | 各节以问题/场景开头？ | 违规=0 |
| P2 | 直接引语密度？ | ≥1处/2000字 |
| P3 | 对称排比？ | =0 |
| P4 | 注水段？ | =0 |

### C层检查（Critic-L2执行）

| # | 检查项 | 阈值 |
|---|--------|------|
| C1 | 好消息/坏消息+明确表态？ | 每章≥1处 |
| C2 | 章尾末句含动词+具体对象？ | 100% |
| C3 | 节间字数标准差≥均值30%？ | 达标 |
| C4 | 整数百分比占数据点比例？ | ≤20% |

### B层检查（Critic-L3 + Proofer-3执行）

| # | 检查项 | 阈值 |
|---|--------|------|
| B1 | 公式化标题？ | =0 |
| B2 | 段落字数变异系数？ | ≥0.3 |
| B2 | 标点类型数？ | ≥5种/千字 |
| B2 | 连续雷同结构段？ | ≤2段 |

### V1 视觉密度检查（S层自动执行）

| # | 检查项 | 阈值 | 说明 |
|---|--------|------|------|
| V1 | 视觉元素密度 | 每5000字≥1个 | Mermaid/表格/流程块/插图标记均算；详见 `templates.md` §八 |

## §2 章节质量报告模板

### 精简模式（每章自动输出）

```
📖 第X章就绪 | 福帮手质量评分：X/10 | [仅不通过项展示]
```

全通过时不展示明细。

### 详细模式（用户说"自检"时输出）

```markdown
## 福帮手质量报告 — 第X章

### S层·句级
| 项 | 结果 | 说明 |
|----|------|------|
| S1-S6 | ✅/❌ | [具体数据] |

### P层·段级
| 项 | 结果 | 说明 |
|----|------|------|
| P1-P4 | ✅/❌ | [具体位置] |

### C层·章级
| 项 | 结果 | 说明 |
|----|------|------|
| C1-C4 | ✅/❌ | [具体位置] |

### V1·视觉密度
| 项 | 结果 | 说明 |
|----|------|------|
| V1 | ✅/❌ | 本章X字，视觉元素X个（密度 1:X字） |

📊 福帮手质量评分：X/10
🔖 初审 ✓  复审 ✓  终审 ✓
```

## §4 典型失败模式速查

| 模式 | 示例 | 违反规则 | 修正方向 |
|------|------|---------|---------|
| 套话开场 | "在数字化浪潮中..." | S1 | 人物/场景/数据开场 |
| 概念先行 | "XX是一种..." | P1 | 问题/困境开场 |
| 排比堆砌 | "首先...其次...最后..." | P3 | bullet list或表格 |
| 万能叙事 | "AI将彻底改变..." | C1 | 好消息/坏消息+表态 |
| 注水重述 | "换言之...也就是说..." | P4 | 删除重述段 |
| 抽象结尾 | "综上所述..." | C2 | 具体下一步行动 |
| 连接词胶水 | "此外...另外...同时..." | S4 | 靠逻辑衔接 |
| buzzword | "赋能底座打通闭环" | S5 | 用日常用语替代 |
| 结构克隆 | 每节300字三段论 | C3 | 长短交错，结构变化 |
| 公式标题 | "XX的重要性" | B1 | 提问/结论/悬念式标题 |
| 整数数据 | "提升50%" | C4 | 47.3%或标注来源 |
| 破折号滥用 | 每段一个"——" | S6 | ≤1次/千字 |
| 分号串联 | "提效率；降成本；改体验" | S6 | 句号断开 |
| 无立场对冲 | "一方面...另一方面..." | C1 | 加"所以我建议..." |
| 节奏平坦 | 连续5个200字段落 | B2 | 长短穿插 |
| 感叹号轰炸 | "太重要了！必须做！" | S6 | 全文≤2个 |
